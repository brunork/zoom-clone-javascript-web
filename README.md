# Zoom Clone 

Zoom clone using javascript | #SemanaJSExpert

### Installation

Install the dependencies and devDependencies and start the application.

```sh
$ cd zoom-clone-javascript-web
$ npm install
$ npm start
```

For the server:

```sh
$ cd server
$ npm install
$ npm run dev
```
